RANBRIDGE Services - React Version
This is a React version of the RANBRIDGE Services website, converted from the original HTML/CSS/JavaScript implementation. The website showcases internship opportunities and courses in various tech fields.

Features
Modern React Architecture: Built with React 18 and functional components
Styled Components: CSS-in-JS styling with styled-components
React Router: Client-side routing for navigation
Framer Motion: Smooth animations and transitions
Responsive Design: Mobile-first approach with responsive layouts
Interactive Elements: Hover effects, loading screens, and particle animations
Components Structure
src/

│   ├── Navbar.js              # Navigation bar with scroll effects

│   ├── ServiceCard.js         # Individual service card component
│   ├── CourseSection.js       # Course details modal/section

│   ├── Terms.js               # Terms of service page
│   ├── Footer.js              # Footer component
│   ├── LoadingScreen.js       # Loading screen animation

│   └── Particles.js           # Floating particles effect
├── index.js                   # React app entry point
└── App.js                     # Main app component with routing
